# 👗 Myntra Frontend Clone

A responsive frontend clone of the Myntra e-commerce website built using **HTML5**, **CSS3**, and **JavaScript**. This project recreates the look and feel of Myntra's interface, focusing on modern UI design, responsive layouts, and interactive web elements.

## 🚀 Features

* 🛍️ Responsive homepage design
* 🔍 Navigation bar with search functionality
* 👕 Product listings and categories
* ❤️ Wishlist and shopping bag icons
* 🎨 Interactive user interface using JavaScript
* 📱 Mobile-friendly and responsive layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Myntra-Clone/

```

## 🎯 Purpose

This project was created to strengthen frontend web development skills by recreating the interface of a popular fashion e-commerce website. It demonstrates responsive web design, DOM manipulation, and interactive user interface development using HTML, CSS, and JavaScript.

## 📌 Note

This is a frontend-only educational project created for learning purposes. It is not affiliated with, endorsed by, or associated with Myntra. No backend functionality, user authentication, or payment processing is included.
